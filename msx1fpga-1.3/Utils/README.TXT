makeKmp.py - Utility to read file definitions of keymaps and make a binary file to the MSX1FPGA core read and adapt key map accordingly.

Use:
python makeKmp.py <keymap.txt> <keymap.kmp>

-------------------------------------------------
decChrBin - Utility to read a MSX character binary (from ROM) and generate a visualization image from all chars

Use:
python decChrBin.py <file.bin>
